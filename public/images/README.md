# Images Directory

This directory contains images used across the Renaissance Kids homeschool hub.

## Solfege Staircase Hero Image

**File:** `rk-solfege-share.png`

**Purpose:** Social media share image (Open Graph and Twitter Card) for the Solfege Staircase game.

**Specifications:**
- Format: PNG
- Recommended size: 1200x630px (1.91:1 aspect ratio)
- Used in: `/homeschool-hub/solfege-staircase/index.html`

**Referenced in HTML meta tags:**
```html
<meta property="og:image" content="https://www.renkids.org/images/rk-solfege-share.png">
<meta name="twitter:image" content="https://www.renkids.org/images/rk-solfege-share.png">
```

To add the image:
1. Place your hero image at `public/images/rk-solfege-share.png`
2. Ensure the image is optimized for web (compressed)
3. Verify it displays correctly when the page is shared on social media
